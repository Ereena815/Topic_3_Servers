ACTIVITY 1

IoT Apps, January 19, 2022

An application that connects two or more devices to the internet allowing communication between them is called an IoT Application. These applications use machine learning algorithms to assess real-time sensor data in order to make decisions such as delivering alerts to users or triggering automated fixes.

Main features of IoT Applications:

  • Provides access to real-time data
  • Good communication between various devices and the application
  • Secure and protected
  • Data base for storage of data
  • Good user interface

ACTIVITY 2

Smart Building, January 19, 2022

Anything which can make everyday work become more efficient, thus, reducing human intervention can be considered smart.

Main features of Smart Buildings:

  • Performs tasks autonomously
  • Predictive maintenance, hence, reduced costs
  • Convenient to use
  • Saves money and time
  • Highly secure
  • Environment friendly

Examples of Smart Building Scenarios:

  • Motion controlled lighting
  • Smart temperature control systems
  • Home assistants like Alexa and Google Home
  • Smart home security
  • Water leakages monitoring systems
  • Air quality monitoring systems
  • Smoke detection

ACTIVITY 3

Control Interface, January 19, 2022

High-level features of a Control Interface:

  • User interactive
  • A good physical control interface
  • Facilitates communication between devices and the internet
  • Can process data efficiently
  • Good connectivity with the user, e.g., sending alerts and triggering some autonomous actions
  • Good performance even under massive data traffic

Parts of a Control Interface:

  • Connectivity with sensors
  • Access to cloud
  • Access to network
  • Machine learning algorithms for data analysis
  • Database for data storage

Users of a Control Interface:

  • Consumers that use the IoT device
  • Consumers that have permission to make changes to the IoT device
  • Providers who own the IoT application
